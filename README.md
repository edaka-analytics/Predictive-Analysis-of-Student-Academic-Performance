Objective
This project explores and models student academic performance using statistical and predictive analysis techniques. The goal is to understand relationships between subjects and build a regression model to predict academic outcomes.
Tools Used
- Python
- pandas
- matplotlib
- scikit-learn
- Jupyter Notebook
Methodology
1. Data loading and cleaning
2. Descriptive statistical analysis
3. Correlation analysis
4. Feature engineering (average score & performance levels)
5. Linear regression modelling
6. Model evaluation using R² and MSE
Key Findings
- Strong positive correlation between reading and writing scores.
- Academic subjects are statistically interconnected.
- Regression model explains a significant proportion of variance in math scores.
- Average score variable provides improved academic performance measurement.
